# 博客系统

## prompt

```
开发一个完整的个人博客系统，该系统需包含后台登录和前台显示两大核心功能模块。技术栈要求如下：前端采用Vue框架结合Vite构建工具，并使用tailwindcss实现UI界面；后端采用Go 语言，数据库使用 sqlite存储，支持 markdown 编写博客文章，支持复制粘贴上传图片
```

## 使用的模型

- glm-5.1
- minimax-2.7
- kimi-k2.5
